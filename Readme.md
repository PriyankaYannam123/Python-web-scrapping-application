# Python Web Scraping Application

## Description:
This Python application **scrapes product prices** from e-commerce websites using **BeautifulSoup** and **requests** libraries. It collects pricing data and stores it in a **MySQL database**. This helps in **real-time price tracking** and comparison for decision-making.

## Technologies Used:
- Python
- BeautifulSoup (for scraping)
- Requests (for HTTP requests)
- MySQL (for database storage)
- MySQL Connector (for connecting Python with MySQL)

## Features:
- Scrapes product prices from e-commerce websites
- Stores scraped data into a MySQL database
- Uses **BeautifulSoup** to parse HTML and extract relevant data

## How to Run:
1. Clone the repository.
2. Set up a **MySQL database** and create the `prices` table.
3. Install the required libraries using:
   ```bash
   pip install beautifulsoup4 requests mysql-connector-python
